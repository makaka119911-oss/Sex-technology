# Сайт для сексологов Татьяны Солнечной и Виктории Румянцевой

Профессиональный сайт для практикующих сексологов с современным дизайном и полным функционалом для привлечения клиентов.

## Структура проекта

- `index.html` - Главная страница
- `training.html` - Страница тренинга "Осознанная сексуальность"
- `about.html` - Страница "О нас" с информацией о специалистах
- `services.html` - Страница с перечнем услуг
- `blog.html` - Блог с фильтрацией по темам
- `contacts.html` - Страница контактов с формой обратной связи
- `style.css` - Основной файл стилей
- `script.js` - JavaScript для функциональности сайта

## Особенности

### Дизайн
- Профессиональный веб-дизайн с тёмно-коралловой цветовой гаммой
- Адаптивная верстка для всех устройств
- Современные UI-компоненты и анимации

### Функциональность
- Возрастная проверка (18+)
- Интеграция с Telegram для отправки заявок
- Интерактивные формы обратной связи
- Фильтрация статей в блоге
- FAQ с аккордеоном
- Плавная прокрутка и навигация

### Контент
- Полное описание тренинга с модулями
- Информация о специалистах с образованием и специализацией
- Подборка полезных статей по темам
- Различные форматы услуг и цены

## Настройка

### Telegram интеграция
Для работы форм обратной связи необходимо:

1. Убедиться, что бот имеет доступ к отправке сообщений
2. Проверить правильность TELEGRAM_BOT_TOKEN и TELEGRAM_CHAT_ID в файле `script.js`

### Контактная информация
Заменить в файлах:
- Номера телефонов
- Email адреса
- Ссылки на социальные сети
- Адрес офиса

### Изображения
Заменить placeholder изображения на реальные фотографии специалистов и тематические изображения.

## Развертывание

Просто загрузите все файлы на хостинг. Сайт готов к работе без дополнительной настройки.

## Технологии

- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Montserrat, Playfair Display)

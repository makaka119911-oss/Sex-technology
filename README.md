# Sexology Website - TSVR\n\n## Описание\nПрофессиональный сайт для студии сексологии и образовательных услуг. Разработан с использованием современных веб-технологий для создания эффективного онлайн-присутствия.\n\n## Структура проекта\n\n```\nSex-technology/\n├── index.html                # Главная страница\n├── training.html             # Страница обучения\n├── about.html                # О студии\n├── services.html             # Услуги\n├── blog.html                 # Блог\n├── contacts.html             # Контакты\n├── css/\n│   ├── style.css            # Основные стили\n│   └── responsive.css        # Адаптивные стили\n├── js/\n│   ├── main.js              # Основной функционал\n│   ├── age-verification.js   # Проверка возраста\n│   └── telegram-bot.js       # Интеграция Telegram\n├── images/\n│   ├── heroes/              # Изображения героев\n│   ├── team/                # Фото команды\n│   └── icons/               # Иконки\n└── README.md\n```\n\n## Особенности\n\n- **Адаптивный дизайн** - Работает на всех устройствах\n- **Модальное окно возраста** - Проверка при входе\n- **Интеграция Telegram** - Отправка сообщений через Telegram Bot API\n- **Современный UI** - Красивый и удобный интерфейс\n- **SEO оптимизация** - Подготовлено для поисковых систем\n\n## Установка\n\n1. Склонируйте репозиторий\n```bash\ngit clone https://github.com/makaka119911-oss/Sex-technology.git\n```\n\n2. Откройте `index.html` в браузере\n\n3. Для Telegram интеграции\n   - Получите токен от BotFather\n   - Добавьте токен в `js/telegram-bot.js`\n   - Добавьте Chat ID\n\n## Установка Telegram Bot\n\n1. Откройте @BotFather в Telegram\n2. Создайте нового бота\n3. Получите токен\n4. Обновите значения в `telegram-bot.js`:\n   - `this.token` - ваш токен\n   - `this.chatId` - ID вашего чата\n\n## Развертывание\n\n- **GitHub Pages** - Автоматическое развертывание с ветки main\n- **Vercel** - Поддержка статического хостинга\n- **Netlify** - Простое развертывание\n\n## Технологии\n
- HTML5\n- CSS3 (Grid, Flexbox, CSS Variables)\n- Vanilla JavaScript\n- Telegram Bot API\n- Unsplash API для изображений\n\n## Контакты\n

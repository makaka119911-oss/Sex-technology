# Спецификация: отслеживание заказа

## Типичные сценарии

1. **Личный кабинет** — список заказов с статусами: `Новый` → `Собран` → `Передан в доставку` → `В пути` → `Доставлен`.
2. **Гостевой трекинг** — ввод номера заказа + телефона или трек-номера ТК на отдельной странице.
3. **Уведомления** — SMS/email со ссылкой на статус (вне scope фронта без бэкенда уведомлений).

## Минимальный UX для лендинга

- Страница успеха показывает **номер заказа** и ссылку «Мои заказы» (`/cabinet`).
- В кабинете — история для авторизованного пользователя из БД; для гостя — сохранённые локально номера заказов + опциональный поиск по номеру и телефону (server action).

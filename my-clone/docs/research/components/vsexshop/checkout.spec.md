# Спецификация: оформление заказа (checkout)

## Обзор

Многошаговый или одностраничный процесс: **контакты → доставка → оплата** (у нас в лендинге — упрощённо: контакты + доставка + подтверждение).

## Поля (типичный набор)

| Группа | Поля |
|--------|------|
| Контакт | Имя, телефон, email |
| Адрес | Город, улица, дом, квартира (индекс — для Почты России) |
| Комментарий | Свободный текст курьеру / упаковке |

## Блоки интерфейса

1. **Способ доставки** — радио-группа: курьер, ПВЗ (СДЭК / Boxberry), Почта России.
2. **Стоимость доставки** — отображается рядом с методом или в итоговом блоке после «расчёта» (на реальном сайте — ответ API ТК).
3. **Итог** — сумма товаров, доставка, к оплате/к получению.
4. **Дискретность** — чекбокс «нейтральная / дискретная упаковка» (часто включён по умолчанию).

## Подтверждение

После отправки — редирект на страницу успеха с **номером заказа** и краткими данными по доставке.

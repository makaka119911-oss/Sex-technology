# Спецификация: корзина (референс UX vsexshop-подобных магазинов)

## Обзор

- **Интеракции:** добавление из каталога и карточки товара, изменение количества `+` / `−`, удаление строки, очистка (опционально).
- **Состояние:** обычно хранится в сессии или `localStorage` до оформления; на сервер синхронизируется при подтверждении заказа.
- **Отображение:** мини-корзина (dropdown / drawer) в шапке + полная страница `/cart` с таблицей/списком позиций, ценой за единицу, промежуточной суммой по строке, общей суммой.

## Ключевые элементы

1. **Карточка строки заказа:** превью фото, название, цена, счётчик количества, иконка удаления.
2. **Пересчёт:** при изменении количества мгновенно обновляются суммы без перезагрузки.
3. **Пустое состояние:** текст «Корзина пуста», ссылка в каталог.
4. **CTA:** кнопка «Перейти к оформлению» ведёт на `/checkout`.

## Поведение

- Добавление дубликата товара увеличивает `quantity`, а не создаёт вторую строку (типичный паттерн).
